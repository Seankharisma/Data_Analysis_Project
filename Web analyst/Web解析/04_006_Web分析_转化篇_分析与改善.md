# 做转化的分析

<p>把握网页的成功（转化率）是有必要设定适合网页目的的目标的。<br>
比如明确的目的有「购入」、「咨询」、「请求资料」、「会员注册」。没有设定转化目标的话，一定要根据网络的目标来设定。
</p>
